Please follow these Instructions for proper execution
  1. I am using adjacency matrix graph representation. Algorithm used Floyd Warshall(2 Reasons; All source shortest path. Negative weight support)
  2. Enter the edge weights to compute. If there is no edge kindly use -. Enter spaces between weights
  3. Sample Input 1
    0 5
    - 0
    This represents a directed edge between vertex 0 and 1 with weight 5. from vertex 1 to 0 there is no vertex. Use - if there is no vertex.
  4. Sample Input 2
    0 5
    5 0
    This represents a undirected edge between vertex 0 and 1 with weight 5.
